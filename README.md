# Zoo Explorer: Fun Animal Facts

A kid-friendly, modern static webpage featuring common zoo animals and fun facts, built for GitHub Pages.

## Local preview

Open `index.html` in your browser.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. In **Settings → Pages**, ensure **Source** is set to **GitHub Actions**.
3. Push to the `main` branch.
4. After the workflow completes, your site will be available at:

   `https://<your-github-username>.github.io/<your-repository-name>/`
